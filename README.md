# Creative Writing AI Agent ✨

This project is a simple AI agent built using **Python**, **Google ADK (Agent Development Kit)**, and **Gemini API**.

The agent helps users generate **creative and motivational stories** from simple prompts. It demonstrates the basic workflow of building AI agents using Google's ADK framework.

---

## 🚀 Features

* Generates creative story ideas
* Provides motivational content
* Demonstrates basic AI agent workflow
* Built using Google ADK and Gemini model
* Easy to run and modify

---

## 🧠 Technologies Used

* Python
* Google ADK (Agent Development Kit)
* Gemini AI Model
* python-dotenv

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/your-username/creative-writing-ai-agent.git
```

Move to the project folder:

```
cd creative-writing-ai-agent
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## 🔑 Setup API Key

Create a `.env` file and add your Gemini API key:

```
GOOGLE_GENAI_USE_VERTEXAI=False
GOOGLE_API_KEY=your_api_key_here
```

---

## ▶️ Run the Agent

Start the ADK web interface:

```
adk web
```

Then open your browser and go to:

```
http://127.0.0.1:8000
```
